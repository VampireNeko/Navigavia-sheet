# Navigavia-sheet

Scheda personaggio digitale per il GdR [Navigavia](https://www.needgames.it/giochi/navigavia/) di Needful Games.

Navigavia è un marchio e un gioco di proprietà di Need Games. Questo progetto è un fan tool non ufficiale e non affiliato. Lo scopo è semplicemente offrire una versione più interattiva della [scheda ufficiale](https://www.needgames.it/wp-content/uploads/2023/11/NAVIGAVIA_Scheda-PG-edit.pdf) da usare durante le nostre sessioni di gioco tra amici.

## Funzionalità
- Gestione attributi, status, risorse
- Import/export JSON
- Salvataggio automatico

## Uso
Apri `index.html` nel browser. Nessuna installazione richiesta.
